# Premiere Capabilities

The following capabilities define what the host API supports:

| Capability | Supported | Fallback UI |
| :--- | :--- | :--- |
| `projectPath` | Yes | Shows error: Cannot detect project path. |
| `activeSequenceRead` | Yes | Shows warning: Cannot read active sequence. |
| `sequenceInventoryRead` | Yes | Shows warning: Cannot read sequence inventory. |
| `trackClipRead` | Yes | Shows warning: Cannot read track clips. |
| `mediaReferenceRead` | Yes | Shows warning: Cannot read media references. |
| `saveEventHooks` | Yes | Shows warning: Cannot hook into save events. |
